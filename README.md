# Crypto Tracker Dashboard

Ik heb dit project gemaakt met Vite + React je kunt hiermee live de coin markt in de gaten
houden en de live prijzen zien.

# Installatie 

## Clone de repository
Open je terminal en plak de volgende regel om het project te downloaden:

git clone https://github.com/Kane026/Crypto-Dashboard.git
Ga naar de projectmap
Navigeer naar de map die zojuist is aangemaakt:

cd Crypto-Dashboard

## Installeer de benodigde pakketten (Dependencies)
Omdat de map node_modules nooit op GitHub staat, moet je deze zelf installeren op basis van het package.json bestand:

npm install

Dit kan even duren, omdat hij nu alle bibliotheken zoals React, HeroUI en Recharts downloadt.

## Start de Development Server
Zodra de installatie klaar is, kun je de app lokaal opstarten:

npm run dev
Bekijk de app
In je terminal verschijnt nu een link (meestal http://localhost:5173). Klik hierop of kopieer dit naar je browser om je dashboard te zien.
